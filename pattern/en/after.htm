Tasks to complete after generating the BPM Integration Service pattern

After generating the BPM Integration Service pattern instance, complete the following steps:

  1. Implement the service operation by using a message flow.

    For information about developing message flows, see Developing Message Flows.

  2. Add the service to a broker archive file (BAR).
    1. Click File > New > BAR file.
    2. Enter the name of your service or select one from the displayed list. The list is filtered to show only projects in the active working set.
    3. Enter a name for the BAR file that you are creating.
    4. Click Finish. A file with a .bar extension is created and is displayed in the Navigator view, under the Broker Archives folder. The Content Editor for the BAR file opens.
    5. On the Prepare tab, select the generated service to add to your BAR file.
    6. Click Build and Save to create your BAR file.
  3. Optional: Configure the BAR file you have created. You are not required to configure the message flow in the BAR file before you deploy the message flows for your BPM Integration Service pattern instance. Later, you might want to change some of the default options depending on your environment.
  4. Deploy the BAR file to an integration server (execution group).
  5. In the Application Development navigator, right-click on Integration Service Description in your service and click Export Integration Service > IBM Business Process Manager. The Select Deployed Service window opens.
  6. You can change the host name or listener port for the service, or accept the defaults, and click Next. The Export IBM Business Process Manager Snapshot window opens.
  7. Enter a name and file path for the new snapshot and click Finish. The snapshot is saved as a new .twx file.
  8. Using the BPM toolkit that was used to create the BPM integration service, import the new .twx file.The BPM integration service is updated with the details of the IBM Integration Bus service.

    9. The BPM integration service can now call the IBM Integration Bus service.
