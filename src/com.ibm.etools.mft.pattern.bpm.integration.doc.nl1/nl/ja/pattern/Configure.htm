「Business Process Manager 統合サービス」パターン・インスタンスを構成するためのタスク

「BPM 統合サービス」パターン・インスタンスを作成するには、以下のステップを実行します。

  1. パターン・エクスプローラーから「BPM 統合サービス」パターンを選択します。
  2. パターン・インスタンスの名前を入力します。
  3. 「OK」をクリックします。 「YourPatternInstanceName - パターン構成」という新しいパネルが開きます。
  4. 「パターン・パラメーターの構成」タブを開きます。
  5. 「パターン・パラメーター」パネルで、「成果物」を展開し、次のフィールドに入力します。
    • IBM Integration Bus サービスによって実装される BPM 統合サービスを選択します。
    • BPM 統合サービスを実装する新規 IBM Integration Bus サービスの名前を入力します。
  6. 「パターン・パラメーターの構成」タブで「生成」をクリックします。

「BPM 統合サービス」パターンのインスタンスが作成されます。
