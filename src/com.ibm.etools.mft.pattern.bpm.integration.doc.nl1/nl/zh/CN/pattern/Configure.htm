用于配置 Business Process Manager Integration Service 模式实例的任务

要创建 BPM Integration Service 模式实例，完成下列步骤：

  1. 从“模式资源管理器”中选择 BPM Integration Service 模式。
  2. 输入模式实例的名称。
  3. 单击确定。将打开一个名为“YourPatternInstanceName - 模式配置”的新面板。
  4. 打开“配置模式参数”选项卡。
  5. 在“模式参数”面板中，展开“工件”并填写下列字段：
    • 选择要由 Integration Bus 服务实现的 BPM 集成服务。
    • 输入新 IBM Integration Bus 服务的名称以实现 BPM 集成服务。
  6. 单击“配置模式参数”选项卡中的生成。

会创建一个 BPM Integration Service 模式的实例。
