套用 BPM Integration Service 型樣之前要完成的作業

在使用此型樣之前，請讀取 IBM Integration Bus 資訊中心內 IBM Business Process Manager 的相關資訊；請參閱使用 IBM Business Process Manager

套用 BPM Integration Service 型樣之前請完成下列作業：

  1. 檢查此型樣是否符合您的商業需求。如需相關資訊，請參閱型樣概觀中的「解決方案」區段。
  2. 如需使用 BPM Integration Service 型樣的限制，請參閱限制。
  3. 在 BPM 工具箱中建立僅包含預設開始事件與結束事件元件的整合服務。
    • 將輸入與輸出變數新增至整合服務。
    • 建立工具箱的 Snapshot。
    • 將 Snapshot 匯出至 BPM 匯出檔（.twx 檔）。

您已備妥套用型樣。
