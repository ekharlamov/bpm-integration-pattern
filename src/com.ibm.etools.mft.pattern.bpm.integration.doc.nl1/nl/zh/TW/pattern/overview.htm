Business Process Manager Integration Service 型樣

您可以使用此型樣來將 IBM Business Process Manager (BPM) 整合服務與 IBM Integration Bus 服務相整合。

解決方案

您可以使用此型樣來產生可由 IBM Business Process Manager 呼叫的新 IBM Integration Bus 服務。

在使用此型樣之前，請讀取 IBM Integration Bus 資訊中心內 IBM Business Process Manager 的相關資訊；請參閱使用 Business Process Manager

BPM 和 IBM Integration Bus 開發者協作完成下列步驟，以將 IBM BPM 整合服務與 IBM Integration Bus 服務相整合：

首先，BPM 開發者必須完成下列作業：

  1. 使用 BPM 工具箱，建立或識別需要呼叫 IBM Integration Bus 服務的 BPM 整合服務。
  2. 建立包含一或多個整合服務（已定義輸入與輸出變數）的 BPM 匯出檔（.twx 檔）。

下一步，IBM Integration Bus 開發者必須完成下列作業：

  1. 從 .twx 檔選取必要的 BPM 整合服務，來產生新的型樣實例。

    所產生的 IBM Integration Bus 服務包含一個服務介面（WSDL 檔），該介面的作業包含對應於 BPM 整合服務定義的輸入及輸出。

  2. 開發 IBM Integration Bus 服務，將服務部署到整合節點（分配管理系統），並驗證該服務運作正確。
  3. 將服務匯出為新的 .twx 檔。

    4. 新的 .twx 檔包含新服務的詳細資料，形式為實作包含服務 URL 與對映商業物件對映的 BPM Web 服務。

最後，BPM 開發者必須完成下列作業：

  1. 使用用來建立 BPM 整合服務的 BPM 工具箱，匯入新的 .twx 檔。

    2. BPM 整合服務將更新為 IBM Integration Bus 服務的詳細資料。現在，BPM 整合服務將呼叫 IBM Integration Bus 服務。
