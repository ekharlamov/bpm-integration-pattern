所有型樣參數會與其預設值一起列出。

標頭為 M 的直欄指出內容是否為必要的。標頭為 C 的直欄指出內容是否為可配置的，這表示您可以在將訊息流程新增至 IBM Integration Bus 保存檔 (BAR) 時變更值。

如果未指出任何預設值，則預設值是空字串。
