配置 Business Process Manager Integration Service 型樣實例的作業

若要建立 BPM Integration Service 型樣實例，請完成下列步驟：

  1. 從「型樣瀏覽器」中選取 BPM Integration Service 型樣。
  2. 鍵入型樣實例的名稱。
  3. 按一下確定。即會開啟名為「YourPatternInstanceName - 型樣配置」的新畫面。
  4. 開啟「配置型樣參數」標籤。
  5. 在「型樣參數畫面」中，展開「構件」並完成下列欄位：
    • 選取要由 IBM Integration Bus 服務實作的 BPM 整合服務。
    • 輸入新的 IBM Integration Bus 服務名稱，以實作 BPM 整合服務。
  6. 按一下「配置型樣參數」標籤中的產生。
