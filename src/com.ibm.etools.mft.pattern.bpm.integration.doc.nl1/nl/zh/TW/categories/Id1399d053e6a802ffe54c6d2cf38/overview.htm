BPM Integration

環境定義

IBM Business Process Manager (BPM) 提供企業層級的商業程序設計與建模，而 IBM Integration Bus 提供企業強度連線與整合。

問題

當業務分析師使用 IBM BPM 開發一個商業程序設計 (BPD) 時，他們可能需要使用企業服務的部分商業資料。企業服務的擁有者可以告訴業務分析師必須提供哪些資訊才能擷取必要的商業資料。

業務分析師瞭解如何定義 BPD 必需的輸入及輸出商業物件，但不知道如何配置與企業服務的連線。例如，企業服務可能使用特定的傳輸類型，可能提供的資料超過所需，也可能要求安全性及特定的服務品質。

選擇指引

當 IBM BPM 使用者已在 BPM 整合服務中定義介面，且該介面必須使用 IBM Integration Bus 服務來實作時使用此型樣。
