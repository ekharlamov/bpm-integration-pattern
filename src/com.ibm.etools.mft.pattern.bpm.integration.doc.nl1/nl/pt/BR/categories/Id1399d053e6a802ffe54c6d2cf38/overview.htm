Integração do BPM

O Contexto

O IBM Business Process Manager (BPM) fornece design e modelagem do processo de negócios em nível corporativo; o IBM Integration Bus fornece forte integração e conectividade corporativas.

O Problema

Quando um analista de negócios desenvolve um Business Process Design (BPD) usando o IBM BPM, ele pode requerer alguns dados de negócios que estão disponíveis a partir de um serviço corporativo. O proprietário do serviço corporativo pode dizer ao analista de negócios quais informações devem ser fornecidas para a recuperação dos dados de negócios necessários.

O analista de negócios sabe como definir os objetos de negócios de entrada e saída necessários para o BPD, mas não sabe como configurar a conectividade para o serviço corporativo. Por exemplo, o serviço corporativo pode usar um tipo de transporte específico, pode fornecer mais dados do que o necessário ou pode requerer segurança ou aderência a uma qualidade de serviço específica.

Orientação de Seleção

Use esse padrão quando um usuário do IBM BPM tiver definido uma interface em um serviço de integração do BPM e essa interface tiver que ser implementada com o uso de um serviço IBM Integration Bus.
