Restrições no Uso do Padrão Integração de Serviço do BPM

É possível usar o padrão BPM Service Integration apenas se você satisfizer todas as seguintes restrições:

  • Um serviço de integração do BPM pode ser usado para gerar apenas um novo serviço IBM Integration Bus; ele não pode ser incluído como uma operação em um serviço IBM Integration Bus existente.
  • O nome da instância padrão deve ser diferente do nome de serviço gerado.
