Tarefas a serem concluídas depois da geração do padrão do Serviço de Integração do BPM

Após a geração da instância padrão do BPM Integration Service, conclua as etapas a seguir:

  1. Implemente a operação de serviço usando um fluxo de mensagens.

    Para obter informações sobre como desenvolver fluxos de mensagens, consulte Desenvolvendo Fluxos de Mensagens.

  2. Inclua o serviço em um arquivo bar (BAR).
    1. Clique em Arquivo > Novo > Arquivo BAR.
    2. Insira o nome do serviço ou selecione um na lista exibida. A lista é filtrada para mostrar apenas projetos no conjunto de trabalhos ativo.
    3. Insira um nome para o arquivo BAR sendo criado.
    4. Clique em Concluir. Um arquivo com uma extensão .bar é criado e exibido na visualização do Navegador, na pasta Broker Archives. O Editor de Conteúdo do arquivo BAR é aberto.
    5. Na guia Preparar, selecione o serviço gerado para incluir no arquivo BAR.
    6. Clique em Construir e Salvar para criar seu arquivo BAR.
  3. Opcional: Configure o arquivo BAR criado. Você não é obrigado a configurar o fluxo de mensagens no arquivo BAR antes de implementar os fluxos de mensagens para a instância padrão do BPM Integration Service. Posteriormente, talvez você queira alterar algumas opções padrão, dependendo do seu ambiente.
  4. Implemente o arquivo BAR em um servidor de integração (grupo de execução).
  5. No navegador Desenvolvimento de Aplicativo, clique com o botão direito do mouse em Descrição do Serviço de Integração em seu serviço e clique em Exportar Serviço de Integração > IBM Business Process Manager. A janela Selecionar Serviço Implementado é aberta.
  6. É possível alterar o nome do host ou a porta do listener para o serviço, ou aceitar os padrões, e clicar em Avançar. A janela Exportar Captura Instantânea do IBM Business Process Manager é aberta.
  7. Insira um nome e um caminho de arquivo para a nova captura instantânea e clique em Concluir. A captura instantânea é salva como um novo arquivo .twx.
  8. Usando o kit de ferramentas do BPM que foi usado para criar o serviço de integração do BPM, importe o novo arquivo .twx. O serviço de integração do BPM é atualizado com os detalhes do serviço IBM Integration Bus.

    9. O serviço de integração do BPM agora pode chamar o serviço IBM Integration Bus.
