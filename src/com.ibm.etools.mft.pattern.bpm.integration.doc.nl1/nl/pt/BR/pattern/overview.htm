Padrão de Serviço de Integração do Business Process Manager

É possível usar este padrão para integrar um serviço de integração do IBM Business Process Manager (BPM) a um serviço IBM Integration Bus.

Solução

É possível usar este padrão para gerar um novo serviço IBM Integration Bus que possa ser chamado pelo IBM Business Process Manager.

Antes de usar este padrão, leia as informações sobre o IBM Business Process Manager no centro de informações do IBM Integration Bus; consulte Trabalhando com o Business Process Manager

Os desenvolvedores do BPM e do IBM Integration Bus trabalham juntos para integrarem um serviço de integração do IBM BPM a um serviço IBM Integration Bus concluindo as seguintes etapas:

Primeiro, o desenvolvedor do BPM deve concluir as tarefas a seguir:

  1. Usando um kit de ferramentas do BPM, criar ou identificar o serviço de integração do BPM que precisa chamar o serviço IBM Integration Bus.
  2. Criar uma exportação do BPM (arquivo .twx) contendo um ou mais serviços de integração com variáveis de entrada e de saída definidas.

Em seguida, o desenvolvedor do IBM Integration Bus deve concluir as tarefas a seguir:

  1. Gerar uma nova instância padrão selecionando o serviço de integração do BPM necessário a partir do arquivo .twx.

    O serviço IBM Integration Bus gerado contém uma interface de serviço (arquivo WSDL) com uma operação com entradas e saídas que correspondem à definição do serviço de integração do BPM.

  2. Desenvolver o serviço IBM Integration Bus, implementá-lo em um nó de integração (broker) e verificar se ele está funcionando corretamente.
  3. Exportar o serviço como um novo arquivo .twx.

    4. O novo arquivo .twx contém detalhes do novo serviço em forma de implementação de uma integração de Serviço da Web do BPM, que inclui a URL do serviço e o mapeamento de objeto de negócios apropriado.

Por fim, o desenvolvedor do BPM deve concluir as tarefas a seguir:

  1. Usando o kit de ferramentas do BPM que foi usado na criação do serviço de integração do BPM, importe o novo arquivo .twx.

    2. O serviço de integração do BPM é atualizado com os detalhes do serviço IBM Integration Bus. O serviço de integração do BPM agora pode chamar o serviço IBM Integration Bus.
