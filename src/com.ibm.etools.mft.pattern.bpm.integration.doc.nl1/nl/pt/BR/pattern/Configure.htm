Tarefas para configurar a instância padrão Serviço de Integração do Business Process Manager

Para criar uma instância padrão do Serviço de Integração do BPM, conclua as seguintes etapas:

  1. Selecione o padrão Serviço de Integração do BPM no Explorer de Padrões.
  2. Digite um nome para a instância padrão.
  3. Clique em OK. Um novo painel chamado YourPatternInstanceName - Configuração de Padrão é aberto.
  4. Abra a guia Configurar Parâmetros Padrão.
  5. No painel Parâmetros Padrão, expanda Artefatos e complete os campos a seguir:
    • Selecione o serviço de integração do BPM a ser implementado pelo serviço IBM Integration Bus.
    • Insira um nome para um novo serviço IBM Integration Bus para implementar o serviço de integração do BPM.
  6. Clique em Gerar na guia Configurar Parâmetros Padrão.

Uma instância do padrão Serviço de Integração do BPM é criada.
