Integración de BPM

El contexto

IBM Business Process Manager (BPM) proporciona diseño y modelado de procesos empresariales a nivel de empresa e IBM Integration Bus proporciona conectividad e integración de fortaleza de la empresa.

El problema

Cuando un analista de negocio desarrolla un diseño del proceso empresarial (BPD) utilizando IBM BPM podría necesitar algunos datos empresariales que están disponibles a partir de un servicio empresarial. El propietario del servicio empresarial puede indicar al analista de negocio qué información debe proporcionarse para recuperar los datos empresariales necesarios.

El analista empresarial sabe cómo definir los objetos de negocio de entrada y salida necesarios para BPD pero no sabe cómo configurar la conectividad con el servicio corporativo. Por ejemplo, el servicio de empresa podría utilizar un tipo de transporte específico, proporcionar más datos de los necesarios, o requerir la seguridad o el cumplimiento de una calidad de servicio específica.

Orientación sobre la selección

Utilice este patrón cuando un usuario de IBM BPM haya definido una interfaz en un servicio de integración de BPM y esa interfaz debe implementarse utilizando un servicio de IBM Integration Bus.
