Servicio de integración de BPM Sí Sí  

El servicio de integración de BPM que será implementado por el servicio de Integration Bus.
Información de servicio
Parámetro de patrón O C Valor predeterminado Descripción
Nombre de servicio de Integration Bus Sí Sí  

Un diálogo del asistente para crear un nuevo servicio o utilizar un servicio existente para implementar el servicio de integración de BPM.

Todos los parámetros de patrón aparecen en la lista con sus valores predeterminados.

La columna con la cabecera O indica si la propiedad es obligatoria. La columna con la cabecera C indica si la propiedad es configurable, lo que quiere decir que puede cambiar el valor al añadir el flujo de mensajes al archivo archivador de IBM Integration Bus (BAR).

Si no se indica el valor predeterminado, éste es una serie vacía.
