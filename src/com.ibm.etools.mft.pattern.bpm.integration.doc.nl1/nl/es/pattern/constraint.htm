Restricciones en el uso del patrón Servicio de integración de BPM

Puede utilizar el patrón Servicio de integración de BPM sólo si se cumplen todas las restricciones siguientes:

  • Un servicio de integración de BPM se puede utilizar para generar sólo un nuevo servicio de IBM Integration Bus; no se puede añadir como una operación a un servicio existente de IBM Integration Bus.
  • El nombre de instancia de patrón debe ser distinto del nombre de servicio generado.
