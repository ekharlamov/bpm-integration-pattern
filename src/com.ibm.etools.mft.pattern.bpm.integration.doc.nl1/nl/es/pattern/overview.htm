Patrón Servicio de integración de Business Process Manager

Puede utilizar este patrón para integrar un servicio de integración de IBM Business Process Manager (BPM) con un servicio de IBM Integration Bus.

Solución

Puede utilizar este patrón para generar un nuevo servicio de IBM Integration Bus que IBM Business Process Manager puede llamar.

Antes de utilizar este patrón, lea la información sobre IBM Business Process Manager en el Information Center de IBM Integration Bus; consulte Trabajar con Business Process Manager

Los desarrolladores de BPM e IBM Integration Bus colaboran para integran un servicio de integración de IBM BPM con un servicio de IBM Integration Bus realizando los pasos siguientes:

En primer lugar, el desarrollador de BPM debe completar las tareas siguientes:

  1. Mediante un kit de herramientas de BPM, cree o identifique el servicio de integración de BPM que tenga que llamar al servicio de IBM Integration Bus.
  2. Cree un archivo de exportación de BPM (archivo .twx) que contenga uno o varios servicios de integración con variables de entrada y salida definidas.

A continuación, el desarrollador de IBM Integration Bus debe realizar las tareas siguientes:

  1. Generar una nueva instancia de patrón seleccionando el servicio de integración de BPM necesario del archivo .twx.

    El servicio de IBM Integration Bus generado contiene una interfaz de servicio (archivo WSDL) que tiene una operación con entradas y salidas que corresponden a la definición del servicio de integración de BPM.

  2. Desarrollar el servicio de IBM Integration Bus, desplegar el servicio en un nodo de integración (intermediario) y verificar que el servicio funciona correctamente.
  3. Exportar el servicio como un nuevo archivo .twx.

    4. El nuevo archivo .twx contiene detalles del nuevo servicio en forma de una implementación de una integración de servicio web BPM que incluye el URL de servicio y la correlación de objeto de negocio correspondiente.

Por último, el desarrollador de BPM debe realizar las tareas siguientes:

  1. Mediante el kit de herramientas de BPM que se utilizó para crear el servicio de integración de BPM, importe el nuevo archivo .twx.

    2. El servicio de integración de BPM se actualiza con los detalles del servicio de IBM Integration Bus. Ahora, el servicio de integración de BPM puede llamar al servicio de IBM Integration Bus.
