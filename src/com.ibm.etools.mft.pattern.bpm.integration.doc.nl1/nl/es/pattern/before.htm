Tareas que deben completarse antes de aplicar el patrón Servicio de integración de BPM

Antes de utilizar este patrón, lea la información sobre IBM Business Process Manager en el Information Center de IBM Integration Bus; consulte Trabajar con IBM Business Process Manager

Realice las tareas siguientes antes de aplicar el patrón Servicio de integración de BPM:

  1. Compruebe que este patrón cumple los requisitos empresariales. Para obtener más información, consulte la sección de soluciones para el patrón en la Visión general de patrones.
  2. Consulte Restricciones para conocer las limitaciones en el uso del patrón Servicio de integración de BPM.
  3. En el Kit de herramientas de BPM, cree un servicio de integración que sólo contenga los componentes predeterminados Suceso de inicio y Suceso de finalización.
    • Añada variables de entradas y salidas al servicio de integración.
    • Cree una instantánea del kit de herramientas.
    • Exporte la instantánea a una exportación BPM (archivo .twx).

Ya está preparado para aplicar el patrón.
