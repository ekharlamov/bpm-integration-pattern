Tareas para configurar la instancia de patrón Servicio de integración de Business Process Manager

Para crear una instancia de patrón Servicio de integración de BPM, realice los pasos siguientes:

  1. Seleccione el patrón Servicio de integración de BPM en el Explorador de patrones.
  2. Escriba un nombre para la instancia de patrón.
  3. Pulse Aceptar. Se abrirá un panel nuevo llamado Nombre_instancia_patrón – Configuración de patrón.
  4. Abra el separador Configurar parámetros de patrón.
  5. En el panel Parámetros de patrón, expanda Artefactos y complete los campos siguientes:
    • Seleccione el servicio de integración de BPM que será implementado por el servicio de IBM Integration Bus.
    • Entre un nombre para un nuevo servicio de IBM Integration Bus para implementar el servicio de integración de BPM.
  6. Pulse Generar en el separador Configurar parámetros de patrón.

Se crea una instancia del patrón Servicio de integración de BPM.
