Tareas que deben completarse después de generar el patrón Servicio de integración de BPM

Después de generar la instancia del patrón Servicio de integración de BPM, realice los pasos siguientes:

  1. Implemente la operación del servicio mediante un flujo de mensajes.

    Para obtener información sobre el desarrollo de flujos de mensajes, consulte Desarrollo de flujos de mensajes.

  2. Añada el servicio a un archivo archivador de intermediario (BAR).
    1. Pulse Archivo > Nuevo > Archivo BAR.
    2. Escriba el nombre de su servicio o seleccione uno en lista visualizada. La lista se filtra para mostrar sólo los proyectos del conjunto de trabajo activo.
    3. Especifique un nombre para el archivo BAR que está creando.
    4. Pulse Finalizar. Se crea un archivo con una extensión .bar y se muestra en la vista Navegador, bajo la carpeta Archivadores de intermediario. Se abre el Editor de contenido para el archivo BAR.
    5. En el separador Preparar, seleccione el servicio generado para añadirlo al archivo BAR.
    6. Pulse Compilar y guardar para crear el archivo BAR.
  3. Opcional: Configure el archivo BAR que ha creado. No es necesario que configure el flujo de mensajes en el archivo BAR antes de desplegar los flujos de mensajes para la instancia de patrón Servicio de integración de BPM. Más adelante, es posible que desee cambiar algunas de las opciones predeterminadas en función de su entorno.
  4. Despliegue el archivo BAR en un servidor de integración (grupo de ejecución).
  5. En el navegador de Desarrollo de aplicaciones, pulse el botón derecho del ratón en Descripción de servicio de integración en el servicio y pulse Exportar servicio de integración > IBM Business Process Manager. Se abre la ventana Seleccionar servicio desplegado.
  6. Puede cambiar el nombre de host o el puerto de escucha para el servicio, o bien aceptar los valores predeterminados, y pulse Siguiente. Se abre la ventana Exportar instantánea de IBM Business Process Manager.
  7. Entre un nombre y vía de acceso de archivo para la nueva instantánea y pulse Finalizar. La instantánea se guarda como un nuevo archivo .twx.
  8. Utilización del kit de herramientas BPM que se utilizó para crear el servicio de integración BPM, importe el nuevo archivo .twx. El servicio de integración de BPM se actualiza con los detalles del servicio de IBM Integration Bus.

    9. Ahora, el servicio de integración de BPM puede llamar al servicio de IBM Integration Bus.
