Tous les paramètres du modèle sont répertoriés avec leur valeurs par défaut.

La colonne avec l'en-tête M indique si la propriété est obligatoire. La colonne avec l'en-tête C indique si la propriété est configurable, ce qui signifie que vous pouvez modifier la valeur lorsque vous ajoutez le flux de messages au fichier d'archive IBM Integration Bus (BAR).

Lorsqu'aucune valeur n'est définie, la valeur par défaut est une chaîne vide.
