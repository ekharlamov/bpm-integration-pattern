Contraintes d'utilisation du modèle Intégration de service BPM

N'utilisez le modèle Intégration de service BPM que si toutes les contraintes suivantes sont respectées :

  • Un service d'intégration BPM peut être utilisé uniquement pour générer un nouveau service IBM Integration Bus. Il ne peut pas être ajouté en tant qu'opération à un service IBM Integration Bus existant.
  • Le nom de l'instance du modèle doit être différent de celui du service généré.
