Tâches à effectuer pour configurer l'instance du modèle Service d'intégration de Business Process Manager

Pour créer une instance de modèle Service d'intégration BPM, procédez comme suit :

  1. Sélectionnez le modèle Service d'intégration BPM dans l'explorateur de modèles.
  2. Entrez un nom à l'instance de modèle.
  3. Cliquez sur OK. Un nouveau panneau nommé YourPatternInstanceName - Configuration du modèle s'affiche.
  4. Ouvrez l'onglet Configurer les paramètres de modèle.
  5. Dans le panneau Paramètres du modèle, développez Artefacts et renseignez les zones suivantes :
    • Sélectionnez le service d'intégration BPM à implémenter par le service IBM Integration Bus.
    • Entrez un nom pour le nouveau service IBM Integration Bus permettant d'implémenter le service d'intégration BPM.
  6. Cliquez sur Générer dans l'onglet Configurer les paramètres de modèle.

Une instance du modèle de service d'intégration BPM est créée.
