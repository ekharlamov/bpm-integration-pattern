Modèle Service d'intégration de Business Process Manager

Ce modèle permet d'intégrer un service d'intégration IBM Business Process Manager (BPM) à un service IBM Integration Bus.

Solution

Ce modèle permet de générer un nouveau service IBM Integration Bus qui peut être appelé par IBM Business Process Manager.

Avant d'utiliser ce modèle, prenez connaissance des informations relatives à IBM Business Process Manager contenues dans le centre de documentation IBM Integration Bus, voir Working with Business Process Manager

Les développeurs BPM et IBM Integration Bus travaillent ensemble pour intégrer un service d'intégration IBM BPM à un service IBM Integration Bus en procédant comme suit :

Le développeur BPM doit tout d'abord exécuter les tâches suivantes :

  1. A l'aide d'un kit d'outils BPM, créer ou identifier le service d'intégration BPM devant appeler le service IBM Integration Bus.
  2. Créer un export BPM (fichier .twx) contenant un ou plusieurs services d'intégration avec des variables d'entrée et de sortie définies.

Le développeur IBM Integration Bus doit ensuite exécuter les tâches suivantes :

  1. Générer une nouvelle instance de modèle en sélectionnant le service d'intégration BPM requis à partir du fichier .twx.

    Le service IBM Integration Bus généré contient une interface de service (fichier WSDL) qui comporte une opération avec des entrées et des sorties correspondant à la définition du service d'intégration BPM.

  2. Développer le service IBM Integration Bus, le déployer sur un noeud d'intégration (courtier) et vérifier qu'il fonctionne correctement.
  3. Exporter le service en tant que nouveau fichier .twx.

    4. Le nouveau fichier .twx contient des détails sur le nouveau service sous la forme d'une implémentation d'une intégration BPM Web Service incluant l'adresse URL du service et un mappage d'objet métier approprié.

Enfin, le développeur BPM doit exécuter les tâches suivantes :

  1. A l'aide du kit d'outils BPM utilisé pour créer le service d'intégration BPM, importer le nouveau fichier .twx.

    2. Le service d'intégration BPM est mis à jour avec les détails du service IBM Integration Bus. Il peut maintenant appeler le service IBM Integration Bus.
