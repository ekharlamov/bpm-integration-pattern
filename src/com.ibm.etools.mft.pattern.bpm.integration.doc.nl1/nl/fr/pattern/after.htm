Tâches à exécuter après avoir généré le modèle Service d'intégration BPM

Après avoir généré l'instance de modèle Service d'intégration BPM, procédez comme suit :

  1. Implémentez l'exploitation de services à l'aide d'un flux de messages.

    Pour plus d'informations sur le développement de flux de messages, voir Developing Message Flows.

  2. Ajoutez le service à un fichier d'archive de courtier (BAR).
    1. Cliquez sur Fichier > Nouveau > Fichier BAR.
    2. Entrez le nom du service ou sélectionnez-en un dans la liste affichée. La liste est filtrée pour contenir uniquement les projets dans l'ensemble actif de projets.
    3. Entrez le nom du fichier BAR à créer.
    4. Cliquez sur Terminer. Un fichier portant l'extension .bar est créé et s'affiche dans la vue Navigateur, sous le dossier Archives de courtier. L'éditeur de contenu du fichier BAR s'affiche.
    5. Dans l'onglet Préparation, sélectionnez le service généré à ajouter au fichier BAR.
    6. Cliquez sur Compiler et sauvegarder pour créer le fichier BAR.
  3. Facultatif : Configurez le fichier BAR que vous avez créé. Vous n'êtes pas tenu de configurer le flux de messages dans le fichier BAR avant de déployer les flux de messages pour l'instance du modèle Service d'intégration BPM. Vous pourrez ensuite, le cas échéant, modifier certaines options par défaut, selon votre environnement.
  4. Déployez le fichier BAR sur un serveur d'intégration (groupe d'exécution).
  5. Dans le navigateur de développement d'application, cliquez avec le bouton droit de la souris sur Description du service d'intégration dans votre service et cliquez sur Exporter le service d'intégration > IBM Business Process Manager. La fenêtre Select Deployed Service s'affiche.
  6. Vous pouvez modifier le nom d'hôte ou le port d'écoute du service ou accepter les valeurs par défaut, puis cliquer sur Suivant. La fenêtre Exportation d'image instantanée IBM Business Process Manager s'affiche.
  7. Entrez un nom et un chemin de fichier pour la nouvelle image instantanée et cliquez sur Terminer. L'image instantanée est enregistrée en tant que nouveau fichier .twx.
  8. A l'aide du kit d'outils BPM utilisé pour créer le service d'intégration BPM, importez le nouveau fichier .twx. Le service d'intégration BPM est mis à jour avec les détails du service IBM Integration Bus.

    9. Il peut maintenant appeler le service IBM Integration Bus.
