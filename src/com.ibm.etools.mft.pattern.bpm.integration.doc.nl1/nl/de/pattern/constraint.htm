Einschränkungen bei der Verwendung des Musters BPM Service Integration

Das Muster BPM Service Integration kann nur verwendet werden, wenn alle der folgenden Bedingungen erfüllt sind:

  • Eine BPM-Serviceintegration kann zurzeit nur zur Erstellung eines neuen IBM Integration Bus-Service verwendet werden, nicht jedoch als zusätzliche Operation zu einem bereits vorhandenen IBM Integration Bus-Service.
  • Der Name der Musterinstanz muss sich von dem generierten Servicenamen unterscheiden.
