Tasks zur Konfiguration der Musterinstanz 'Business Process Manager-Integrationsservice'

Führen Sie zur Erstellung einer Instanz des Musters 'BPM Integration Service' die folgenden Schritte aus:

  1. Wählen Sie im Musterexplorer das Muster 'BPM Integration Service' aus.
  2. Geben Sie einen Namen für die Musterinstanz ein.
  3. Klicken Sie auf OK. Ein neues Fenster mit dem Titel Musterinstanzname - Musterkonfiguration wird geöffnet.
  4. Öffnen Sie die Registerkarte 'Musterparameter konfigurieren'.
  5. Erweitern Sie in diesem Fenster für die Musterparameter den Eintrag 'Artefakte' und füllen Sie die folgenden Felder aus:
    • Wählen Sie den BPM-Integrationsservice aus, der vom IBM Integration Bus-Service implementiert werden soll.
    • Geben Sie einen Namen für einen neuen IBM Integration Bus-Service ein, um den BPM-Integrationsservice zu implementieren.
  6. Klicken Sie auf der Registerkarte 'Musterparameter konfigurieren' auf Generieren.

Eine Instanz des Musters 'BPM Integration Service' wird erstellt.
