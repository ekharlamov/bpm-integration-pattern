Tasks vor der Anwendung des Musters 'BPM Integration Service'

Lesen Sie vor der Verwendung dieses Musters die Informationen zu IBM Business Process Manager im Information Center von IBM Integration Bus; siehe Mit IBM Business Process Manager arbeiten

Führen Sie vor der Anwendung des Musters 'BPM Integration Service' die folgenden Tasks aus:

  1. Prüfen Sie, ob dieses Muster Ihren Geschäftsanforderungen entspricht. Weitere Informationen finden Sie im Abschnitt 'Lösung' für das Muster unter Musterübersicht.
  2. Lesen Sie vor Verwendung des Musters 'BPM Integration Service' den Abschnitt zu den Einschränkungen.
  3. Erstellen Sie im BPM-Toolkit einen Integrationsservice, der nur die Standardkomponenten Startereignis und Endereignis enthält.
    • Fügen Sie dem Integrationsservice Eingabe- und Ausgabevariablen hinzu.
    • Erstellen Sie eine Momentaufnahme des Toolkits.
    • Exportieren Sie die Momentaufnahme in eine BPM-Exportdatei (Erweiterung .twx).

Sie können das Muster nun anwenden.
