Tasks to configure the Business Process Manager Integration Service Pattern instance

To create a BPM Integration Service pattern instance, complete the following steps:

  1. Select the BPM Integration Service pattern from the Patterns Explorer.
  2. Type a name for the pattern instance.
  3. Click OK. A new panel opens named YourPatternInstanceName - Pattern Configuration.
  4. Open the Configure Pattern Parameters tab.
  5. In the Pattern Parameters Panel, expand Artifacts and complete the following fields:
    • Select the BPM integration service to be implemented by the IBM Integration Bus service.
    • Enter a name for a new IBM Integration Bus service to implement the BPM integration service.
  6. Click Generate in the Configure Pattern Parameters tab.

An instance of the BPM Integration Service pattern is created.
