Tasks to complete before applying the BPM Integration Service pattern

Before using this pattern, read the information about IBM Business Process Manager in the IBM Integration Bus information center; see Working with IBM Business Process Manager

Complete the following tasks before applying the BPM Integration Service pattern:

  1. Check that this pattern meets your business requirements. For more information, see the Solution section for the pattern in the Pattern Overview.
  2. See Constraints for restrictions on using the BPM Integration Service pattern.
  3. In the BPM Toolkit, create an integration service that contains only the default Start Event and End Event components.
    • Add inputs and outputs variables to the integration service.
    • Create a snapshot of the toolkit.
    • Export the snapshot to a BPM export (.twx file).

You are ready to apply the pattern.
